Objetivo deste microserviço é gerar um banco com embbedings para comparar textos através de similaridade
Para se gerar o dataset necessário é necessario seguir os passos a seguir

Objetivo macro do algoritimo

1º Gerar embbedings através do endereço http://10.1.1.95:5020/generateembeddings?split=train
    Como dica de otimização a primeira que é gerado se o banco tiver muitos idsiguais pode se chamar api para ela e começar denovo pois o sistema só ira processar textos unicos

2º Encontrar ids altamente semelhantes através do endereço  http://10.1.1.95:5020/generateembeddings?split=train

3º Encontrar ids colidentes ou seja altamente conflitantes que tenham algo grau de similaridade chamando a api  http://187.85.181.118:18020/generate_ids_colidentes

4º Gerar a base final com o split final em http://10.1.1.95:5020/generateembeddings?split=final
